## 7. 方块硬度（hardness）与挖掘
硬度（破坏基准，-1 = 不可破坏）：
| 方块 | 硬度 |
|---|---|
| 泥土 dirt / 沙子 sand | 0.5 |
| 草方块 grass / 沙砾 gravel | 0.6 |
| 石头 stone | 1.5 |
| 圆石 / 橡木板 / 橡木原木 | 2.0 |
| 煤矿石 coal_ore | 3.0 |
| 黑曜石 obsidian | 50 |
| 基岩 bedrock | -1（不可破坏） |

**挖掘时间公式**（秒）：
```
canHarvest = 用了能正常掉落的工具？
multiplier = canHarvest ? 1.5 : 5
seconds = hardness × multiplier / toolSpeed
# 等价：每 tick 进度 = toolSpeed / hardness / (canHarvest?30:100)，进度满 1 即破坏
# 徒手 toolSpeed = 1
```
例：石头 + 木镐(speed 2，可掉落) → 1.5×1.5/2 ≈ **1.13 s**；石头徒手 → 1.5×5/1 = **7.5 s**（且不掉落）。
